# v0.1.0

## Highlights

- Added one-click installers for GLM Coding Plan + Claude Code on macOS and Windows.
- Added one-click installers for OpenAI Codex CLI on macOS and Windows.
- Added cleanup scripts for removing the older managed GLM Node runtime.
- Added bilingual README, security guidance, and contribution docs.

## Installation

### macOS

- `scripts/macos/install-glm-claude.command`
- `scripts/macos/install-openai-codex.command`

### Windows

- `scripts/windows/install-glm-claude.bat`
- `scripts/windows/install-openai-codex.bat`

## Verification

After installation, verify the expected command exists:

- `claude --version`
- `coding-helper --version`
- `codex --version`
- `node --version`
- `npm --version`

## How to use

- Run the GLM installer if you want Claude Code configured for GLM Coding Plan.
- Run the Codex installer if you want Codex CLI configured for an OpenAI-compatible gateway.
- Run the cleanup script only after the newer system-level installation is confirmed working.
