# v0.1.1

## Highlights

- Simplified release assets so the download list is easier to understand.
- Added user-facing suffixes such as `【双击执行】` and `【下载后双击执行】`.
- Packaged Windows installers as zip bundles so users only need one download.
- Kept the repository file layout clean while making release downloads more beginner-friendly.

## Installation assets

### macOS

Choose files ending with:

- `【双击执行】.command`

### Windows

Choose files ending with:

- `【下载后双击执行】.zip`

Each Windows zip bundle includes both the clickable `.bat` launcher and the required `.ps1` script.

## Notes

GitHub will still show:

- `Source code (zip)`
- `Source code (tar.gz)`

These are GitHub-generated source archives, not the recommended installer downloads for end users.
