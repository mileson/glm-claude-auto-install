# v0.1.10

## 这次更新了什么

这次把 4 个安装器一起更新到了当前可分发版本：

- OpenAI Codex macOS 安装器
- OpenAI Codex Windows 安装器
- GLM Claude Code macOS 安装器
- GLM Claude Code Windows 安装器

## 关键改进

- Windows Codex 安装器现在写入 `gpt-5.5`、`responses`、`https://ai.558669.xyz` 这一组中转配置，并补齐 `auth.json`。
- Windows Codex 安装器修复了 npm 全局安装、用户目录、PATH、缓存清理和低磁盘空间提示相关问题。
- Windows GLM Claude Code 安装器优先使用官方 Claude Code bootstrap，并保留 winget / npm 兜底路径。
- macOS Codex 安装器新增图形界面入口、日志、复用已保存 Key、临时 Key 文件和安装完成后的启动入口。
- macOS 构建脚本支持跳过 notarize 的本地打包模式，方便先生成附件和做内容校验。

## 验证结果

- Windows Codex 远程安装后已完成 `WINDOWS_CODEX_OK` 连通验证。
- Windows GLM Claude Code 远程安装后已完成 `WINDOWS_GLM_OK` 连通验证。
- 4 个 zip 附件均已校验包内脚本与仓库源脚本一致。
