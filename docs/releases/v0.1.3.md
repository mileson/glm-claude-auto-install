# v0.1.3

## 这次更新了什么

这次主要不是改安装逻辑，而是把页面改得更适合新手直接操作：

- 默认中文说明页改成了“只要 3 步”结构
- 首页直接放了可下载链接，不需要用户自己找文件
- 明确告诉用户不要下载 `Source code`
- 把每个平台应该点哪个文件说得更直白
- 强调“下载到本地后双击执行”这一条主路径

## 新手怎么用

### 如果你要安装 GLM Coding Plan + Claude Code

- macOS：下载 `GLM-Claude-macOS-Double-Click.command`
- Windows：下载 `GLM-Claude-Windows-Download-Then-Double-Click.zip`

### 如果你要安装 OpenAI Codex CLI

- macOS：下载 `OpenAI-Codex-macOS-Double-Click.command`
- Windows：下载 `OpenAI-Codex-Windows-Download-Then-Double-Click.zip`

## 只要 3 步

1. 按系统点击下载对应文件
2. 下载到本地
3. 双击执行，跟着提示输入信息

## 提醒

GitHub 页面里的：

- `Source code (zip)`
- `Source code (tar.gz)`

是自动生成的源码包，不是给普通用户直接安装用的。
