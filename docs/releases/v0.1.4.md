# v0.1.4

## 这次更新了什么

这次把发布下载区进一步精简成更适合新手的形式。

现在主下载区只保留 4 个安装文件：

1. mac 版智谱 Claude Code
2. Windows 版智谱 Claude Code
3. mac 版 OpenAI Codex
4. Windows 版 OpenAI Codex

## 为什么这么改

之前把清理旧版环境的脚本也放进 release 下载区后，新手很容易分不清自己到底该点哪一个。

这次调整后，普通用户只需要看 4 个主文件就够了。

## 如果真的需要清理旧版环境怎么办

清理脚本没有删除，只是不再放到主下载区。

如果以后需要，可以直接到仓库里的 `scripts/` 目录获取。

## 提醒

GitHub 仍然会自动显示：

- `Source code (zip)`
- `Source code (tar.gz)`

这两个是源码包，不是普通用户应该下载的安装器。
