# v0.1.5

## 这次更新了什么

这次主要更新的是下载页说明，让第一次使用的人更容易看懂。

## 我该怎么选

如果你只是想先安装好工具，按下面选就行：

### 我要装智谱 Claude Code

- 我用 mac → 下载 `GLM-Claude-macOS-Double-Click.command`
- 我用 Windows → 下载 `GLM-Claude-Windows-Download-Then-Double-Click.zip`

### 我要装 OpenAI Codex

- 我用 mac → 下载 `OpenAI-Codex-macOS-Double-Click.command`
- 我用 Windows → 下载 `OpenAI-Codex-Windows-Download-Then-Double-Click.zip`

## 现在主下载区只保留 4 个文件

普通用户现在只需要看这 4 个主文件：

1. mac 版智谱 Claude Code
2. Windows 版智谱 Claude Code
3. mac 版 OpenAI Codex
4. Windows 版 OpenAI Codex

这样更适合第一次使用的人直接选择，不容易下载错。

## 提醒

GitHub 仍然会自动显示：

- `Source code (zip)`
- `Source code (tar.gz)`

这两个是源码包，不是普通用户应该下载的安装器。
