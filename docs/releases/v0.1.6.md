# v0.1.6

## 这次更新了什么

这次把 OpenAI Codex 安装器进一步简化了。

现在用户执行安装器时：

- **只需要输入 API Key**
- 不再需要自己填写 Base URL
- 不再需要自己填写模型
- 不再需要自己填写 reasoning effort

## 现在的使用方式

### OpenAI Codex

执行安装器后，只需要：

1. 输入 API Key
2. 等待自动安装完成

其余内容会自动按预设配置写入本地，包括：

- Base URL
- 默认模型
- 默认 reasoning
- 其他常用配置

## 为什么这样改

因为对于普通用户来说：

- 只输入 API Key 最容易理解
- 其他配置如果已经固定，就不应该让用户重复填写
- 这样可以减少填错配置的概率
