# v0.1.7

## 这次更新了什么

这次把 mac 版下载入口从 `.command` 升级成了：

- **已签名**
- **已公证**
- **可直接双击的 `.app` 安装器压缩包**

## 现在 mac 用户怎么用

### 智谱 Claude Code

- 下载：`GLM-Claude-macOS-App-Double-Click.zip`
- 解压后双击里面的 `.app`

### OpenAI Codex

- 下载：`OpenAI-Codex-macOS-App-Double-Click.zip`
- 解压后双击里面的 `.app`

## 为什么这样改

之前的 `.command` 更容易触发 macOS 的安全提示。

现在改成：

- Developer ID 签名
- Apple notarization 公证
- stapler 回写票据

用户双击打开时，体验会明显更顺。

## Windows 没变

Windows 版仍然保持：

- 下载 zip
- 解压
- 双击里面的 `.bat`
