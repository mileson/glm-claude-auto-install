# v0.1.8

## 这次更新了什么

这次把智谱 Claude Code 安装器进一步简化了。

现在执行安装器时：

- **不再让用户选择中国站 / Global**
- 默认直接使用 **中国站**
- 用户只需要输入自己的 API Key

## 为什么这样改

对大多数用户来说：

- 第一次安装时不需要做套餐判断
- 直接固定中国站更容易理解
- 能进一步减少点错和填错的概率

## 现在的体验

### 智谱 Claude Code

用户执行安装器后只需要：

1. 输入 API Key
2. 等待自动安装完成

安装器会自动使用：

- 中国站校验接口
- 中国站 Anthropic 兼容地址
- 默认中文配置
